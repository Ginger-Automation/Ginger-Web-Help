Create New/Open Solution

Solution- A Solution is a logical structure that is a complete automation package in itself that can be loaded and executed in GINGER. The first step to start with GINGER is to create a Solution.

 

A new solution can be created using the ‘Create New Solution’ option.

An existing Solution stored on the PC can be opened by clicking on the name drop-down as shown below and click on the ‘Open Existing solution’ option.

An existing Solution from SVN Repository can be opened by clicking on the name drop-down as shown below and click on the ‘Download Existing solution’ option.
