Introduction

Send Keys action is used to send the keyboard keys to the desired location.

Apart from the regular alphabets and numbers, the below special keys can also be sent:-

Backspace, Break, Delete, Capslock, Down arrow, End, Enter, Esc, Help, Home, Insert, Left, NumLock, Page down, Page up, Right, Scroll lock, Tab, Up, Cntrl, shift, Alt

 

How to Configure Action

Pre-requisite

          A Solution should be created with the agent like ‘web’,’ Java’,’ Mobile’,’ power Builder’ etc.

       

 

       

 

 

Action Operation

 

Double click on ‘Business flow’ and then, click on the ‘Add New (+)’ button shown below, then select the ‘Actions Library’.

          

 

            

 

 

 

Go to the ‘General Actions’ tab and select the ‘Send Keys Action’ then click on that arrow which is highlighted below to add the action or simply drag & drop the element.
‎

 

 

The below screen should be displayed.

Enter the desired location in the ‘Locate Value:’ field by either typing the locator path or using the Spy button.

Spy button is nothing, but it is used for automation. For applying spy first of you need to start the SPY then go to click on that area where you want to spy and then hold the ‘control’ button then that area will turn into a ‘pink’ color.

Enter the required key in the ‘Keys to be sent’

‘Locate By’ dropdown consist of multiple options. You can choose any one that you want.

Click on the Save button to add the action to the Activity Actions.

    

 

 

Run Action will Perform specific options and will give desired results.
