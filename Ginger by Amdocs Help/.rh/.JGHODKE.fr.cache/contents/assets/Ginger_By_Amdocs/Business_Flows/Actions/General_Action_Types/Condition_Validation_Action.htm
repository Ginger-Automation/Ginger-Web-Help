Condition Validation Action     Contents Introduction How to Configure Action   ‎ Introduction  Condition Validation Action is used for creating validation. How to Configure Action              Double click on the Business flow. Click on the ‘+’ icon               On the Action Library screen go to the ‘General Actions’ tab and double click on the ‘Condition Validation Action’                   Condition Validation Action will be added under the Activity as shown below              Double click on it to see the below screen and then go to the ‘Operation Settings’ tab. Enter the condition in the Value field                      The Run Action button will execute the condition given.
