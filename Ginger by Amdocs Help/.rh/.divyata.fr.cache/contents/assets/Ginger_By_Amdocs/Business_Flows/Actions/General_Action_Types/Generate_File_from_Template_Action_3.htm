Generate File from Template Action Introduction This action generates a CSV file from a template. How to create this Action Pre-requisite A Solution should be created with the agent like ‘Web', 'Mobile’ etc. How to create this Action Double click on ‘Business flow’ then click on the ‘Add New (+)’ button shown below.         Go to the ‘Action Library’ and then ‘General Actions’ tab and select the ‘File Transfer Action’ then simply drag and drop it into the respective field or click on that arrow highlighted below.                                The below screen should be displayed. ‘File Action’ dropdown contain ‘CSVFromTemplate’ option. Choose a ‘Data File Name’ using Browse. Enter the name in ‘Data File Name’. Click on Ok to add the action to the Activity Actions. Run Action will create an output file in the respective format.
