Click on the ‘Refresh’ button present on the top right of Window Explorer                          Features Following are the features and tabs that can be used to identify the elements on the page.   Screenshot tab   On this tab, we can select the element from the screenshot instead of moving to the browser window for selecting the element. Properties of each element and locating features can be validated at the same time.            In Actions Tab – User has the option to choose elements on the page                          Users can also integrate the page with existing POM. Select the “Integrated POM” checkbox for selecting POM models.                               Page Source tab This tab can be used to identify the source of the content. Tree View tab This view will list down all the elements displayed on the page in form of a tree and could be useful to understand the structure of the page.   Grid View tab This view would help the user to select the element type and getting its path and other features in a grid format.
