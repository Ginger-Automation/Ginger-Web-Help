File Transfer Action     Contents Introduction How to Configure Action Pre-requisite Action Operation   ‎ Introduction  File transfer action is used to transfer files from the local PC to Citrix server or any other server especially Unix server.     How to Configure Action Pre-requisite             A Solution should be created.   Action Operation Double click on the Business flow. Click on the ‘+’ icon     On the ‘Action Library’ screen go to the ‘General Actions’ tab and double click on the ‘File Transfer Action’                  File Transfer Action will be added under the Activity as shown below   Double click on it and go to the ‘Operation Settings’ tab. Select the desired file using the Browse button near the ‘PC Source file path’. Enter the Unix path, Host, Port, Username, Password, PrivateKeyFile and KeyPassPhrase (If any).     Run Action will transfer the file to the Unix location
