How to add Business Flow in the Runner? ‎ Introduction A Run Set is a set of selected Business flows that are grouped for execution. This is a very powerful feature for parallel execution. A Business Flow has to be created to utilize the Run set features.   How to Create Run Set? Default Runset When a solution is created a default run set is automatically added to it. A Runner, Runner1 is also available by default and is highlighted.   How to add a new Runset? Under the Run tab, click on the + button as shown below at the extreme top right.   In the new window, enter the Runset name and click on the Ok button.   A new Runset is added with a default Runner in it.   Click on the Save button as shown below at the extreme top right   How to select different Run sets? Under the Run tab, click on the open folder icon as shown below at the extreme top right.   In the window as displayed below, select the desired Runset and click on the Select button.     How to shift to another Runset? Click on the folder button as shown below to Open the Runset.   List of saved Runsets will be displayed as shown below a window. Select the desired runset and Click on the Select button.       How to create Runset folders? Click on the folder icon as shown below.                           Right-click on the Run Sets and select the ‘Add Sub Folder’ option as shown below.                             Enter the folder name and click on the Ok button.   The folder will be created as shown below.       How to Copy and Paste Runset? Click on the folder icon as shown below.   Right-click on the desired Runset, select the Copy option.   Right-click on the location where you want to paste the runset and select the Paste option.   What is a Runner? The highlighted frame below is called Runner. The Business Flows are added in this runner       How to add Business Flow in the Runner? Select the desired runner and click on the + button as shown below to add the Business Flow.   The window displays all the Business Flow in the solution. Select the desired one and click on the Select button.   All the Activities and Actions in the Business Flow get added to the Runner.
