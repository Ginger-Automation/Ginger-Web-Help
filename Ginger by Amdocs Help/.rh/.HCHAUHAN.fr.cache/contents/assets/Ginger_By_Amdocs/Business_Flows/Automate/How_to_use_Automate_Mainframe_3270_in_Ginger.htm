How to use Automate Mainframe 3270 in Ginger     Contents Configurations Add Action     Configurations Add a new Target Application with platform Mainframe ‎ ‎ Add a New Agent of Type Mainframe 3270 ‎ ‎ Configure the agent with your application details, Do not change Term Type unless necessary                          Add the Target Application to the desired Business Flow Add a new activity and choose the Target application with Platform Mainframe. On Proper connection, you’ll see your application screen with a message that the connection is successful.                         Add Action Set Text To Mainframe Action   Locate By: Use By Caret Position or BY XY  #x200e ByCaretPosition: Use the caret position shown in the Ginger mainframe window for the desired location. ‎ ByXY: We can also use the XY in the format x,y to set the location of the text ‎Set Text and Send Enter Checkbox: Tick this if you want to send enter after setting text.   Get Details from MainFrame Action ‎Similar to Set Text, Get Details also requires the Locate by and Locate Value. ‎ ‎The value field is the length of the required text starting from the provided Caret Position/ XY. ‎If left blank it will return the text till the next whitespace. ‎   Send Keys to Mainframe Use this Action to send Special Keys to the mainframe like entering, Tab Insert, Function Keys, etc. ‎ Users just need to select the desired key from the dropdown, no other configuration is required. ‎
