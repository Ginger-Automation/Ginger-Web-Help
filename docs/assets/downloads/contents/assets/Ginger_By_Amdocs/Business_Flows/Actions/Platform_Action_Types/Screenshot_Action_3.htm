Introduction

This action is taken a screenshot of the active window and saves it in a folder.

 

How to Configure Action

 Pre-requisite

                A Solution should be created with Agent having Web, Mobile, Windows, PowerBuilder, or Android                               Device Platform.

                

 

 

                

 

 

Action Operation

Double click on the Business flow. In the Activity Actions, click on the ‘Add New (+)’ button as shown below.

Then click on ‘Actions Library’ as shown in the 2nd Screenshot.

             

 

 

                 

 

 

In the ‘Platform Actions’ tab, select the ‘Screenshot Action’ then simply drag and drop into the ‘Actions’ field or click on that arrow which is appeared just above the ‘Action Type’ as shown below.

                   

 

Double click on ‘Screen Shot Action’ to open it.

‘Save to folder Name’ is used to enter the folder path which is selected using the ‘Browser’ button.

Click on the ‘save’ icon as Highlighted below to add the action to the Activity Actions.

 

 

 

Run Action will take a screenshot of the active window in the Web Agent and save it to the given folder.

The screenshot name will be <Date><Time>Screen Shot Action.jpg e.g., ‘12_06_2017_15_46_03_380Screen Shot Action.jpg’ as shown below.
