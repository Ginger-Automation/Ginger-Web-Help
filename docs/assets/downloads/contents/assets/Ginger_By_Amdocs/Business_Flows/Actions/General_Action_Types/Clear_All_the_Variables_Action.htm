‎Introduction

Clear All Variables is an Action that is used before starting the Solution suite to clear all the variables to make them blank before using them.

When a solution is run and values are stored in the variables, these values can create an issue when the solution is run the next time. To avoid such issues this Action is created.

The below variable types of values are made blank: -

Selection list

String

 

How to Configure Action

Pre-requisite

          Before performing this action, a set of variables is required to be created in the Solution.

          The below section can be referred from Ginger Help Library to know how to create variables.

 

           Solution-> Business Flows-> Activity-> Actions Library-> General Actions-> Set Variable Action

 

Action Operation

 

Double click on the Business flow. Click on the ‘+’ icon

            

 

            On the Action Library screen go to the ‘General Actions’ tab and double click on the ‘Clear All Variables Action’

           

 

             Condition Validation Action will be added under the Activity as shown below

 

 

Double click on it to see the below screen and then go to the ‘Operation Settings’ tab. Enter the condition in the Value field

 

 

 

The Run Action button will clear all the values of the Variables described in the Introduction.
