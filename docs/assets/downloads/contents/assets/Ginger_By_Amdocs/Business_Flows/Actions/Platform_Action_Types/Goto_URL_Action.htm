Introduction 

This action is used for opening a URL in Ginger.

 

How to Configure Action

Pre-requisite

A Solution should be created with Agent having Web or Mobile Platform

 

 

 

 

 

Action Operation

 

Double click on the Business flow. In the Activity Actions, click on ‘Add New (+)’ button as shown below and then click on ‘Actions Library’ highlighted in 2nd screen shot.

 

Then In the ‘Platform Actions’ tab, select the ‘Browser Action’ then simply drag and drop into respective field or click on that arrow which is higlighted in below screenshot .

 

 

As shown below, select the Operation Type as ‘Goto URL’.

Enter the URL name in the ‘URL’ field.

Select the option if the website has to be opened in the current agent or new tab or new window.

Click on ‘Save’ button which is highlighted below to add the action to the Activity Actions.

 

 

 

 

 

Run Action will open the URL in the Web Agent.
