What is Environment?

     What is Environment? 

These can be used to store global details that will be used across multiple Business flows present in a solution.

It can be used to store Database connection details, Login Credentials, Unix Server details, or any general parameter one would want to be used across globally.

    Environment Configuration

Go to Resources Tab, click on the Environments and expand it to see the Environment details.

Steps to add Parameter: 

  1. To add Parameter click on the + as shown below.
  2. The same parameter can be added to all the other environments

3. Select the Parameter type from the displayed list. Name is mandatory.

Steps to add Database connection:

1. Click on the Databases tab and then click on the + button as shown below

2. Select the database from the drop down and enter the connection details in the below window.

     Enter the username and password.

3. Click "Use Connection String" to automatically populate the connection string based on the selected database

4. Click on the Test DB connection button to check the connection

     When successful the click on the “Add Database” button to add the database.
